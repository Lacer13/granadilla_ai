# Como abrir el notebook en Google Colab

## Opcion 1: desde el README

1. Entrar al repositorio de GitHub.
2. Presionar el boton "Open in Colab".
3. Esperar a que Google Colab cargue el notebook.

## Opcion 2: desde Google Colab

1. Entrar a Google Colab.
2. Ir a File.
3. Seleccionar Open notebook.
4. Ir a la pestana GitHub.
5. Buscar el repositorio Lacer13/granadilla_ai.
6. Abrir el archivo:

notebooks/yolo_tiny_granadilla_training.ipynb

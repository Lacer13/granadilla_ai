# Registro de capturas

## Objetivo

Registrar cada sesion de captura de imagenes para mantener control del dataset.

## Archivo principal

dataset_management/control_capturas.csv

## Columnas

- fecha: dia de captura.
- responsable: persona que tomo las fotos.
- clase: buena, golpeada, inmadura o prueba.
- cantidad_fotos: numero de imagenes tomadas.
- lugar: ubicacion o ambiente.
- iluminacion: dia, tarde, noche, luz artificial, etc.
- observaciones: problemas, condiciones especiales o comentarios.

## Ejemplo

fecha,responsable,clase,cantidad_fotos,lugar,iluminacion,observaciones
2026-05-21,Luis,buena,30,laboratorio,luz artificial,fondo uniforme
